# Сравнительный анализ Быстрой сортировки и сортировки слиянием
При запуске кода, нужно будет ввести размер массива
Каждый тест на n-ом размере проходит 1000 раз (можно поставить другое значение в переменной - NUM_TESTS)
Время работы алгоритма суммируются в avgQuickTime и в avgMergeTime и потом делеться на NUM_TESTS, чтобы получить среднее время работы
