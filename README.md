# Сравнительный анализ алгоритмов сортировки: Quick Sort vs Merge Sort

## Описание проекта

Программа проводит сравнительный анализ двух алгоритмов сортировки:
- Быстрая сортировка (Quick Sort)
- Сортировка слиянием (Merge Sort)

При запуске программы необходимо ввести размер генерируемого массива. Каждый тест для заданного размера выполняется 1000 раз (количество можно изменить через константу `NUM_TESTS`).

## Алгоритм работы программы

1. Генерация случайных массивов заданного размера
2. Измерение времени выполнения каждого алгоритма
3. Суммирование времени работы в переменных `totalQuickTime` и `totalMergeTime`
4. Расчет среднего времени выполнения: деление суммы на `NUM_TESTS`
5. Вывод результатов сравнительного анализа

## Формат тестирования

- Многократный запуск (1000 итераций по умолчанию)
- Автоматическая генерация тестовых массивов
- Использование высокоточного таймера `std::chrono`
- Статистически значимые результаты благодаря усреднению

## Требования

- Компилятор C++ с поддержкой стандарта C++17 или новее
- Стандартная библиотека C++
- 4 ГБ оперативной памяти (рекомендуется)
- 500 МБ свободного места на диске
